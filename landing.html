See what your agent is doing

Your agent should show its work.

A chat window hands you an answer and throws away the reasoning. A notebook keeps every step it took — as code you can read, re-run, correct, and hand to someone else.

q3-churn-analysis working

persistent

The work doesn't vanish when the conversation scrolls away. It's a document with a URL, still there in six months when someone asks how you got the number.

auditable

Every cell is the actual code that produced the actual output. Nothing is summarized or quietly assumed — you can check the one step you don't believe.

extensible

Disagree with the approach? Edit the cell. Everything downstream recomputes on its own, because a notebook is reactive rather than sequential.

shareable

Send a link, not a screenshot and a paragraph of caveats. Your colleague opens the same live thing you're looking at and can push on it themselves.

quick

No environment, no install, no notebook server to babysit. The agent writes into a page that is already running.

So what is a notebook?

A document that runs.

Prose, data, and code in one place — where changing anything updates everything that depends on it, instantly, in the browser you already have open.

First

Write a little code

JavaScript, SQL, Markdown, HTML — side by side in one document. Pull from a file, a database, or an API without leaving the page.

Then

It recalculates itself

Cells know what they depend on. Move a slider and every chart downstream redraws — nothing to re-run, no stale results.

And

Send the link

What you built is already a web page. Anyone you share it with gets the live version, and can fork it to ask their own question.

Build notes

Palette

Not invented — these are Observable Plot's own categorical colors (#4269d0, #6cc5b0, #efb118, #ff725c). Each band owns one, so the page reads as a chart with four series. Neutrals carry a slight blue bias toward the accent instead of sitting at pure grey.

Stagger

Bands alternate: label left with the chart bleeding right, then mirrored, with opposing vertical offsets so they interlock rather than stack. The alternating background fill is gone — that grey/white banding was what made it read as slabs.

Flourishes

Hairline gridlines at the measure edges, a tick mark per band in its own hue, gradient scroll-axis, masked logo marquee that de-saturates on hover, scroll-reveal, and a rail whose active tick takes the current band's color. All reduced-motion aware.

Numbering

Bands are labelled Series 01–04 rather than 01/02/03 steps. These aren't a sequence and shouldn't pretend to be one — but they are series in a chart, which the color system already says.

Type

Instrument Sans against IBM Plex Mono — deliberately not Inter or Space Grotesk. The mono face carries the band titles, so the technical voice does the naming and the charts do the talking.

Single theme, on purpose

No dark mode. Every Observable embed renders light, and a dark shell would frame six white rectangles badly. The agent blade is the one dark moment, which is where the boldness is spent.
