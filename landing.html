So what is a notebook?

A document that runs.

Observable Notebooks are lightweight computational documents that let you think, explore, and explain with code.

lightweight

Nothing to install

A notebook is a document, not a server. Open the URL and it is already running.

fast

No kernel, no restart

Vanilla JavaScript and TypeScript in the browser you already have open.

reactive

Change one thing

Cells know what they depend on. Move a slider and everything downstream follows.

shareable

Send a link, not a file

It is already a web page. Whoever opens it gets the live thing, and can fork it.

expressive

Visualizations, not just charts

Sankey diagrams, waffle charts, treemaps, advanced maps — not just another bar chart.

ai-native

An artifact an agent can hold

Runtime-aware AI runs the code and inspects live values, so every claim is checkable.
